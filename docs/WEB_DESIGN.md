# Agent Town Web 视觉设计文档

## 1. 设计理念

### 1.1 整体风格
- **类型**: 3D 俯视视角沙盒游戏观察器
- **美学**: 体素风格 (Voxel Art) + 现代简约 UI
- **氛围**: 明亮、友好、清晰易读
- **参考**: Minecraft (体素), Stardew Valley (配色), Townscaper (建筑美学)

### 1.2 设计原则
1. **清晰优先**: 玩家能快速区分地形、建筑、Agent
2. **性能优先**: 支持大规模场景渲染 (1000+ 方块)
3. **一致性**: 所有视觉元素遵循统一的体素网格系统
4. **可扩展性**: 新建筑类型、地形类型能轻松融入

## 2. 3D 渲染规范

### 2.1 坐标系统
```
X轴: 东西方向 (右为正)
Y轴: 高度方向 (上为正)  
Z轴: 南北方向 (下为正)

每个方块: 1×1×1 单位
地面高度: 0 (方块底部在 y=-0.5, 顶部在 y=0.5)
```

### 2.2 地形渲染

#### 基础方块
- **尺寸**: 1×1×1 (长宽高)
- **网格对齐**: 所有方块中心坐标为整数 (x, 0.5, z)
- **堆叠规则**: 高度 N = N 个 1×1×1 方块垂直堆叠
  ```
  高度 0: 1 个方块 (y: 0 到 1)
  高度 1: 2 个方块堆叠
  高度 2: 3 个方块堆叠
  ```

#### 材质
| 类型 | 颜色 | RGB | 说明 |
|------|------|-----|------|
| 草地 | 绿色 | `#5a8f4a` | 主地形，饱和度适中 |
| 泥土 | 褐色 | `#8b7355` | 高地或裸露地形 |
| 水面 | 蓝色 | `#4a90e2` | 透明度 0.8 |
| 岩石 | 灰色 | `#6b7280` | 悬崖或矿区 |

#### 地形边缘
- **无平滑过渡**: 保持体素风格，硬边
- **悬崖**: 高度差 ≥ 2 时显示为陡崖纹理

### 2.3 建筑渲染

#### 基础结构
```
建筑主体: 长方体
  - 宽度: w 格
  - 深度: d 格
  - 高度: h 格
  
位置锚点: 建筑中心点 (x, 0, z)
  - 主体中心: (0, h/2, 0) 相对于锚点
```

#### 屋顶样式 (金字塔型)
```
类型: 四方锥 (Square Pyramid)
旋转: 45° (Y轴)，使边与建筑主体对齐
尺寸: 
  - 底面宽度: max(w, d) × 0.75
  - 高度: h × 0.6
位置: 建筑顶部中心

对齐方式:
  - 旋转 0°: 角对齐 (× 不使用)
  - 旋转 45°: 边对齐 (✓ 使用)
```

#### 建筑类型配色
| 类型 | 墙面 | 屋顶 | 门框 |
|------|------|------|------|
| 政府建筑 | `#8b7355` | `#cd853f` | `#4a3728` |
| 私人住宅 | `#a08060` | `#d2691e` | `#5c4033` |
| 商店 | `#9b8b75` | `#deb887` | `#654321` |
| 工坊 | `#7a6b55` | `#bc8f8f` | `#3d2b1f` |

#### 建筑细节
- **门**: 位于正面中央，0.6×0.8 尺寸
- **窗**: (可选) 墙面嵌入，发光效果
- **烟囱**: (可选) 屋顶一角，细小圆柱

### 2.4 Agent 渲染

#### 基础模型
```
类型: 胶囊体 (Capsule)
尺寸: 
  - 半径: 0.25
  - 高度: 0.5
位置: (x, 0.5, z) - 站立在地面上

动画:
  - 待机: 轻微上下浮动 (正弦波, 振幅 0.05)
  - 移动: (TODO) 朝向移动方向
```

#### 玩家配色
| 角色 | 颜色 | RGB | 标识 |
|------|------|-----|------|
| Alice | 粉红 | `#ff6b6b` | 🔴 |
| Bob | 青色 | `#4ecdc4` | 🔵 |
| 其他 | 黄色 | `#ffe66d` | 🟡 |

#### 状态指示
- **在线**: 头顶绿色小圆点 (半径 0.08)
- **离线**: 灰色半透明
- **战斗中**: 头顶红色闪烁标记
- **选中**: 底部光环光圈

### 2.5 环境渲染

#### 光照
```
环境光:
  - 强度: 0.5
  - 颜色: 白色

方向光 (太阳):
  - 位置: [15, 25, 10]
  - 强度: 1.0
  - 阴影: 开启
  - 阴影贴图: 1024×1024
  - 阴影范围: 60×60
```

#### 网格
```
类型: 无限网格
主网格: 每 5 格一条粗线
次网格: 每 1 格一条细线
颜色:
  - 主线: `#666666`
  - 次线: `#444444`
透明度: 0.5
```

#### 天空
```
类型: 纯色背景
颜色: `#1a1a2e` (深蓝紫色)
或: 渐变色 (可选)
```

## 4. 性能规范

### 4.1 渲染优化
```
地形:
  - 使用 InstancedMesh 批量渲染同类型方块
  - 视锥体裁剪: 只渲染视野内方块
  - LOD: 远距离简化 (可选)

建筑:
  - 合并几何体 (可选)
  - 阴影优化: 仅主要建筑投射阴影

Agent:
  - 限制同屏数量: 最多 50 个
  - 动画使用 GPU (shader)
```

### 4.2 数据更新
```
刷新频率:
  - 位置更新: 每 5 秒
  - 状态更新: 每 10 秒
  - 地形更新: 按需 (建造/破坏时)

长轮询:
  - 事件获取: 30 秒超时
  - 断线重连: 3 次重试
```

## 7. 技术实现要点

### 7.1 使用的库
```javascript
// 核心
@react-three/fiber    // React Three.js 渲染
@react-three/drei     // 辅助组件 (OrbitControls, Grid, Text)
three                  // Three.js 核心

// 状态
zustand               // 状态管理

// 网络
axios                 // HTTP 请求
```

### 7.2 关键组件
```typescript
// 地形渲染 - 使用 InstancedMesh
<Terrain blocks={blocks} />

// 建筑渲染 - 组合几何体
<Building building={building} />

// Agent 渲染 - 带动画
<AgentMarker agent={agent} />

// 主场景
<Canvas shadows camera={{ position: [20, 20, 20], fov: 45 }}>
  <ambientLight intensity={0.5} />
  <directionalLight position={[15, 25, 10]} intensity={1} castShadow />
  <OrbitControls minDistance={10} maxDistance={60} />
</Canvas>
```
