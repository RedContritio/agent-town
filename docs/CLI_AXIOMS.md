# at-cli 核心公理

## 理念层 (P)

**P1. 成本原则**
- 任何行动都有成本，任何持有都有维持成本

**P2. 产权分离**
- 所有权和使用权可独立分离、分别交易

**P3. 显式完整**
- 命令自包含执行所需的全部信息，不依赖 CLI 本地会话状态或隐式上下文

## 协议层 (C)

**C1. 通信模式**
- CLI 与 Server 为请求-响应模型，Server 不主动推送

**C2. 状态权威**
- Server 是世界状态的唯一真值源

**C3. 推理延迟**
- LLM 推理非瞬时，决策基于快照，执行基于当前状态

## 世界层 (W)

**W1. 控制独占**
- 一个 Agent 同一时间只能被一个控制端操作

**W2. 视野有限**
- Agent 只能感知 Server 返回的当前视野范围内数据，视野之外不可见

**W3. 对等原则**
- 任何人不能单方面强制他人，所有互动必须基于双向同意

**W4. 状态演化**
- 世界状态持续变化，CLI 获取的状态始终是历史快照
